#  My MEAN Stack Journey

Hey there! 
This repo is where I’m documenting my journey of learning and building with the **MEAN stack** (MongoDB, Express, Angular, Node.js).

I’ve recently started exploring full-stack development, and this project is my hands-on way of understanding how everything connects — from backend APIs to frontend UI.

---

##  Why I Started This

I wanted to move beyond just theory and actually *build stuff*. MEAN stack felt like a great place to start since it uses JavaScript everywhere.

Also:

* I wanted to understand how real-world apps work
* Learn backend + frontend integration
* Get comfortable with APIs and databases

---

##  Tech Stack

* **MongoDB** – for storing data
* **Express.js** – backend framework
* **React** – frontend 
* **Node.js** – runtime environment

---

## 📁 What’s in This Repo

Right now, it’s a work in progress. You’ll find:

* Basic server setup using Express
* API routes (CRUD operations)
* MongoDB connection setup
* Initial Angular frontend (coming along gradually)

I’ll keep updating this as I learn more.
